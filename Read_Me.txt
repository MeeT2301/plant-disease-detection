

In This Folder, The main python file is index.py, The HTML files are in  tempelates folder, The model is in ml_model folder in pickled state, Upload Images is the folder where it stores uploaded images from the website and static contains files related to css, javascript and bootstrapping.

To run this program, follow these steps:
1. open the command prompt
2. create directory to the plant disease main folder
3. use the command set FLASK_APP=index which compiles the application file
4. Then use the command flask run
5. Now the application will be hosted in the local host server

to see how the project works:
1. open web application in the local server.
2. it will show the home page from which we should upload the images from the upload_images folder.
3. after uplaoding photos from the folder sumbit the image.
4. it will redirect you to the information page about the plant and disease.
